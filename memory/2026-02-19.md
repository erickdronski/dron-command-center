## Lessons
- Claude Code CLI requires auth/PTY confirmation ‚Äî `dangerouslySkipPermissions` in settings.json bypasses warning but still needs trust confirmation per directory
- Fine-grained GitHub PATs (github_pat_) behave differently from classic PATs ‚Äî for repo write + workflow scope, classic PATs are simpler
- GitHub Actions secrets must be encrypted with repo's public key via nacl before pushing via API
- Hardcoded absolute paths in API routes break Vercel ‚Äî always use `process.cwd()`
