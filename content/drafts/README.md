# Post Drafts

This folder contains drafts in progress.

## Naming Convention
`YYYY-MM-DD-post-type-topic.md`

Example: `2026-02-26-strategic-openclaw-trend.md`

## Draft Status

| File | Status | Scheduled |
|------|--------|-----------|
| (empty) | - | - |

## Workflow

1. Create draft from template in CONTENT-ENGINE.md
2. Fill in your specific details/numbers
3. Review and edit
4. Move to ../ready-to-post/ when finalized
